# 🏥 Hospital Appointment System

A responsive **Hospital Appointment Management System** built using
**HTML, CSS, Bootstrap, JavaScript, jQuery, and JSON Server**. The
project allows patients to register, log in, book appointments, and
enables administrators to manage appointments and doctor schedules
efficiently.

## ✨ Features

### 👨‍⚕️ Patient Features

-   Patient registration and login
-   Book appointments with doctors
-   View appointment history
-   Search and filter appointments
-   Profile management

### 🛠️ Admin Features

-   Admin login dashboard
-   View and manage appointments
-   Mark appointments as:
    -   Appointed
    -   Visited
    -   Missed
-   Monitor appointment statistics
-   Manage doctor availability and tokens

### 📊 Appointment Management

-   Daily token generation
-   Token reset for every new day
-   Date-based appointment filtering
-   Status tracking for appointments

## 🧰 Tech Stack

## 🚀 Getting Started

### 1. Clone the repository

``` bash
git clone <repository-url>
cd hospital-appointment-system
```

### 2. Install JSON Server

``` bash
npm install -g json-server
```

### 3. Start the backend server

``` bash
json-server --watch database/Hospital_Appointment_System.json --port 3000
```

### 4. Open the application

Open `Homepage.html` in your browser or use Live Server.
